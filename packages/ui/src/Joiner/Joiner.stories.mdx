# Joiner

`Joiner` component joins multiple React elements using a delimiter which by default is a comma `,`.

```tsx
<Joiner>
  <Typography element="span">Manvendra Singh</Typography>
  <Typography element="span">Keychron K8 Pro</Typography>
  <Typography element="span">Logitech MX 3S</Typography>
</Joiner>
```
